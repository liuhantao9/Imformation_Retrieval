	<h1 id="firstHeading" class="firstHeading" lang="en">Name change</h1>
	
	<div id="bodyContent" class="mw-body-content">

<div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/File:Navneforandring.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/2f/Navneforandring.jpg/220px-Navneforandring.jpg" decoding="async" width="220" height="316" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/2f/Navneforandring.jpg/330px-Navneforandring.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/2/2f/Navneforandring.jpg 2x" data-file-width="404" data-file-height="580" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/File:Navneforandring.jpg" class="internal" title="Enlarge"></a></div>Name change certificate issued by <a href="/wiki/Christian_X_of_Denmark" title="Christian X of Denmark">Christian X of Denmark</a> in 1917</div></div></div>
<p><b>Name change</b> generally refers to the legal act by a person of adopting a new name different from their name at birth, marriage or adoption.
</p><p>The procedures and ease of a name change vary between jurisdictions. In general, <a href="/wiki/Common_law" title="Common law">common law</a> jurisdictions have loose procedures for a name change while <a href="/wiki/Civil_law_(legal_system)" title="Civil law (legal system)">civil law</a> jurisdictions are more restrictive.
</p><p>A <a href="/wiki/Pseudonym" title="Pseudonym">pseudonym</a> is a name that differs from the original or true name and does not require legal sanction. It is generally adopted to conceal a person's identity, but can also be done for personal, social or ideological reasons.
</p>

<h2><span id="Reasons_for_changing_one.27s_name"></span><span class="mw-headline" id="Reasons_for_changing_one's_name">Reasons for changing one's name</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=1" title="Edit section: Reasons for changing one&#039;s name">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul><li>Marriage or civil partnership.</li>
<li>To avoid a <a href="/wiki/Stalker" class="mw-redirect" title="Stalker">stalker</a> or <a href="/wiki/Harassment" title="Harassment">harassment</a></li>
<li>To choose a surname associated with a hobby, interest, or accomplishment (e.g., old name Henry Schifberg, new name <a href="/wiki/Henry_Lizardlover" title="Henry Lizardlover">Henry Lizardlover</a>)</li>
<li>To receive an inheritance conditional on adopting the name of the deceased (this reason was once quite common in landowning families in the UK)</li>
<li>To replace a frivolous name given by their parents (e.g., old name Mikey Maus, new name Marcus Maus)</li>
<li>To replace a name which might be considered undesirable with a more desirable one (e.g., old surname Lipschitz, new surname London)</li>
<li>To dissociate oneself from a former religion (e.g., Ahmed Abdullah to Adel Karim)</li>
<li>To dissociate themselves from a famous or infamous person (e.g., old name <a href="/wiki/Michael_Jackson" title="Michael Jackson">Michael Jackson</a> new name Martin Jackson)</li>
<li>To identify with a famous or infamous person (e.g., old name Simon Johnson, new name Simon <a href="/wiki/King_Arthur" title="King Arthur">Pendragon</a>)</li>
<li>To dissociate themselves from a family <a href="/wiki/Black_sheep_(term)" class="mw-redirect" title="Black sheep (term)">black sheep</a> (e.g., relatives of <a href="/wiki/Adolf_Hitler" title="Adolf Hitler">Adolf Hitler</a>).</li>
<li>To dissociate themselves from an ethnic origin (e.g., changing <a href="/wiki/Battenberg_family" title="Battenberg family">Battenberg</a> to <a href="/wiki/Mountbatten" class="mw-redirect" title="Mountbatten">Mountbatten</a> or Jan Ludvik Hoch to <a href="/wiki/Robert_Maxwell" title="Robert Maxwell">Robert Maxwell</a>).</li>
<li>To revert to a former name (e.g., <a href="/wiki/Cynthia_Lennon" title="Cynthia Lennon">Cynthia Lennon</a>, first wife of <a href="/wiki/John_Lennon" title="John Lennon">John Lennon</a>, after her divorce from later husband John Twist).</li>
<li>Commercial sponsorship (e.g., <a href="/wiki/Jimmy_White" title="Jimmy White">Jimmy White</a> temporarily became Jimmy Brown for <a href="/wiki/HP_Sauce" title="HP Sauce">HP brown sauce</a>,<sup id="cite_ref-BBCBrown1_1-0" class="reference"><a href="#cite_note-BBCBrown1-1">&#91;1&#93;</a></sup> and <a href="/wiki/Ashley_Revell" title="Ashley Revell">Ashley Revell</a> became Ashley blue square Revell for <a href="/wiki/The_Rank_Group" title="The Rank Group">The Rank Group</a>'s Blue Square service)</li>
<li>Protest or activism (e.g., old name Jennifer Taylor, new name Jennifer <a href="/wiki/Environmentalism" title="Environmentalism">Save the Forests</a>)</li>
<li>To change to a fictional character's name, (e.g., old name Tracy Darling, new name <a href="/wiki/Tracy_Beaker" title="Tracy Beaker">Tracy Beaker</a>)</li>
<li>To make their name more attractive or "catchy" so as to increase their chance of success (see <a href="/wiki/Stage_name" title="Stage name">Stage name</a>)</li>
<li>To change legal name to one used in everyday life (e.g., where middle name has been used throughout life)</li>
<li>To <a href="/wiki/Anglicise" class="mw-redirect" title="Anglicise">anglicise</a> a name or choose a different name if one’s name is too difficult to pronounce or spell for most people</li>
<li>To better fit one's <a href="/wiki/Gender_identity" title="Gender identity">gender identity</a> as a <a href="/wiki/Transgender" title="Transgender">transgender</a> person (e.g., old name Bruce Jenner, new name <a href="/wiki/Caitlyn_Jenner" title="Caitlyn Jenner">Caitlyn Jenner</a>)</li></ul>
<h2><span class="mw-headline" id="United_States">United States</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=2" title="Edit section: United States">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In the United States, <a href="/wiki/State_law_(United_States)" title="State law (United States)">state laws</a> regulate name changes. Several federal court rulings have set precedents regarding both court decreed name changes and <a href="/wiki/Common_law" title="Common law">common law</a> name changes (changing the name at will), including <i><a href="/wiki/Lindon_v._First_National_Bank" title="Lindon v. First National Bank">Lindon v. First National Bank</a></i>.
</p><p>Usually a person can adopt any name desired for any reason. As of 2009, 46 states allow a person legally to change names by usage alone, with no paperwork, but a court order may be required for many institutions (such as banks or government institutions) to officially accept the change.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup>  Although the states (<a href="/wiki/Law_of_Louisiana" title="Law of Louisiana">except Louisiana</a>) follow common law, there are differences in acceptable requirements; usually a <a href="/wiki/Court_order" title="Court order">court order</a> is the most efficient way to change names (which would be applied for in a <a href="/wiki/State_court_(United_States)" title="State court (United States)">state court</a>), except at marriage, which has become a universally accepted reason for a name change. It is necessary to <a href="/wiki/Pleading" title="Pleading">plead</a> that the name change is not for a <a href="/wiki/Fraud" title="Fraud">fraudulent</a> or other illegal purpose, such as evading a <a href="/wiki/Lien" title="Lien">lien</a> or <a href="/wiki/Debt" title="Debt">debt</a> or for <a href="/wiki/Defamation" title="Defamation">defaming</a> someone else.
</p><p>Applicants may be required to give a reasonable explanation for wanting to change their names. A fee is generally payable, and the applicant may be required to post legal notices in newspapers to announce the name change. Generally the judge has limited <a href="/wiki/Judicial_discretion" title="Judicial discretion">judicial discretion</a> to grant or deny a change of name, usually only if the name change is for fraudulent, frivolous or <a href="/wiki/Morality" title="Morality">immoral</a> purposes.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup>
In 2004, a <a href="/wiki/Missouri" title="Missouri">Missouri</a> man succeeded in changing his name to They.<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">&#91;4&#93;</a></sup> The <a href="/wiki/Minnesota_Supreme_Court" title="Minnesota Supreme Court">Minnesota Supreme Court</a> ruled that a name change to 1069 could be denied, but that Ten Sixty-Nine was acceptable (<i>Application of Dengler</i>, 1979); the <a href="/wiki/North_Dakota_Supreme_Court" title="North Dakota Supreme Court">North Dakota Supreme Court</a> had denied the same request several years before (<i>Petition of Dengler</i>, 1976).<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">&#91;5&#93;</a></sup>
</p><p>In nearly all states, a person cannot choose a name that is intended to mislead (such as adopting a celebrity's name), that is intentionally confusing, or that incites violence; nor can one adopt, as a name, a <a href="/wiki/Racial_slur" class="mw-redirect" title="Racial slur">racial slur</a>, a threat, or an <a href="/wiki/Obscene" class="mw-redirect" title="Obscene">obscenity</a>.
</p><p>Some examples of typically allowed reasons for name changes in the U.S. include:
</p>
<ul><li>Adopting a new surname upon marriage (typically the surname of the spouse, a <a href="/wiki/Hyphenated_name" class="mw-redirect" title="Hyphenated name">hyphenated surname</a>, or some combination of parts of both surnames).  This is usually done without court proceedings.</li>
<li>Returning to the use of a prior surname (e.g., a <a href="/wiki/Maiden_name" class="mw-redirect" title="Maiden name">maiden name</a>) upon divorce.</li>
<li>Simplification or improved familiarity of spelling or pronunciation.</li></ul>
<p>Under the federal immigration-and-nationality law, when aliens apply for <a href="/wiki/Naturalization" title="Naturalization">naturalization</a>, they have the option of asking for their names to be changed upon the grants of citizenship with no additional fees. This allows them the opportunity to adopt more <a href="/wiki/Cultural_assimilation#Immigrant_name_changing_as_a_form_of_assimilation" title="Cultural assimilation">Americanized</a> names.<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">&#91;6&#93;</a></sup> During the naturalization interview, a petition for a name change is prepared to be forwarded to a <a href="/wiki/United_States_federal_courts" class="mw-redirect" title="United States federal courts">federal court</a>. Applicants certify that they are not seeking a change of name for any unlawful purpose such as the avoidance of debt or evasion of <a href="/wiki/Police" title="Police">law enforcement</a>. Such a name change would become final if within their jurisdiction, once a <a href="/wiki/United_States_federal_courts" class="mw-redirect" title="United States federal courts">federal court</a> naturalizes an applicant.
</p><p>To maintain a person's <a href="/wiki/Identity_(social_science)" title="Identity (social science)">identity</a>, it is desirable to obtain a formal order so there is continuity of personal records.
</p>
<h3><span class="mw-headline" id="Informal_methods_of_legal_name_change">Informal methods of legal name change</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=3" title="Edit section: Informal methods of legal name change">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id="Assumed_name">Assumed name</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=4" title="Edit section: Assumed name">edit</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>The "open and notorious" use of a name is often sufficient to allow one to use an <a href="/wiki/Assumed_name" class="mw-redirect" title="Assumed name">assumed name</a>. In some jurisdictions, a trade name distinct from one's legal name can be registered with a county clerk, <a href="/wiki/Secretary_of_State_(U.S._state_government)" class="mw-redirect" title="Secretary of State (U.S. state government)">secretary of state</a>, or other similar government authority. Persons who wish to publish materials and not to be associated with them may publish under <a href="/wiki/Pseudonym" title="Pseudonym">pseudonyms</a>; such a right is protected under <a href="/wiki/Case_law" title="Case law">case law</a> pursuant to <a href="/wiki/United_States_Constitution" class="mw-redirect" title="United States Constitution">United States Constitution</a>.
</p>
<h4><span class="mw-headline" id="Usage_method">Usage method</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=5" title="Edit section: Usage method">edit</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>A common law name (i.e. one assumed for a non-fraudulent purpose) is a legal name.<sup id="cite_ref-7" class="reference"><a href="#cite_note-7">&#91;7&#93;</a></sup> In most states a statutory method, while quick and definitive, only supplements the common law method,<sup id="cite_ref-8" class="reference"><a href="#cite_note-8">&#91;8&#93;</a></sup> unless the statute makes itself exclusive. Although a person may sue under a common law name.<sup id="cite_ref-9" class="reference"><a href="#cite_note-9">&#91;9&#93;</a></sup>
</p><p>In <a href="/wiki/California" title="California">California</a> the "usage method" (changing the name at will under common law) is sufficient to change the name.  Not all jurisdictions require that the new name be used exclusively.<sup id="cite_ref-10" class="reference"><a href="#cite_note-10">&#91;10&#93;</a></sup> Any fraudulent use or intent, such as changing the name to the same name as another person's name, may invalidate this type of name change.
</p><p>Specifically in California, Code of Civil Procedure § 1279.5 and Family Code § 2082 regulate common law and court decreed name changes. Code of Civil Procedure § 1279.5 (a) reads, "Except as provided in subdivision (b), (c), (d), or (e), nothing in this title shall be construed to abrogate the common law right of any person to change his or her name." Subdivisions b through e preclude one from changing their name by common law if they are in state prison, on probation, on parole, or have been convicted of a serious sex offense. If a person is not in any of these categories, then a common law name change is allowed. (Family Code § 2082 also contains some of the same wording.)
</p>
<h4><span class="mw-headline" id="Preferred_name">Preferred name</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=6" title="Edit section: Preferred name">edit</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>Many universities, hospitals, and other institutions allow one to use a "preferred name" instead of one's legal name.  This name can show up on class rosters, online learning platforms, and student <a href="/wiki/ID_card" class="mw-redirect" title="ID card">ID cards</a>.  It provides a "transitional"
name change for those who have yet to, or cannot, receive a court-ordered name change.<sup id="cite_ref-UC_Davis_11-0" class="reference"><a href="#cite_note-UC_Davis-11">&#91;11&#93;</a></sup><sup id="cite_ref-Leicester_12-0" class="reference"><a href="#cite_note-Leicester-12">&#91;12&#93;</a></sup>
</p>
<h3><span class="mw-headline" id="Official_registration">Official registration</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=7" title="Edit section: Official registration">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>A legal name change is merely the first step in the name-change process. A person must officially register the new name with the appropriate authorities whether the change was made as a result of a court order, marriage, divorce, adoption, or any of the other methods described above. The process includes notifying various government agencies, each of which may require legal proof of the name change and that may or may not charge a fee. Important government agencies to be notified include the <a href="/wiki/Social_Security_Administration" title="Social Security Administration">Social Security Administration</a>,<sup id="cite_ref-13" class="reference"><a href="#cite_note-13">&#91;13&#93;</a></sup> <a href="/wiki/Bureau_of_Consular_Affairs" title="Bureau of Consular Affairs">Bureau of Consular Affairs</a><sup id="cite_ref-14" class="reference"><a href="#cite_note-14">&#91;14&#93;</a></sup> (for passports), the <a href="/wiki/Federal_Communications_Commission" title="Federal Communications Commission">Federal Communications Commission</a>,<sup id="cite_ref-15" class="reference"><a href="#cite_note-15">&#91;15&#93;</a></sup> the <a href="/wiki/Selective_Service_System" title="Selective Service System">Selective Service System</a><sup id="cite_ref-16" class="reference"><a href="#cite_note-16">&#91;16&#93;</a></sup> and the Department of Motor Vehicles (for a new driver's license, learner permit, state identification card, or vehicular registration). Additionally the new name must be registered with other institutions such as employers, banks, doctors, mortgage, insurance and credit card companies. Online services are available to assist in this process either through direct legal assistance or automated form processing.
</p><p>Most states require name changes to be registered with their departments of motor vehicles (DMVs) within a certain amount of time, and some state motor vehicle departments require updated social security cards to make changes, by first registering a new name with the Social Security office:

<p>The fees for registering a new name vary from state to state. The forms, along with the state-specific requirements, can generally be obtained for free.
</p><p>Many states will require reasons for wanting a name change. For example, in Florida, a court will not grant a petition for a change of name if it finds that (i) the petitioner has ulterior or illegal motives in seeking the name change, (ii) the petitioner's civil rights are suspended, or (iii) granting the name change will invade the property rights (<i>e.g.</i>, intellectual property rights) of others.<sup id="cite_ref-29" class="reference"><a href="#cite_note-29">&#91;29&#93;</a></sup>
</p>
<h2><span class="mw-headline" id="United_Kingdom">United Kingdom</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=8" title="Edit section: United Kingdom">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In the <a href="/wiki/United_Kingdom_of_Great_Britain_and_Northern_Ireland" class="mw-redirect" title="United Kingdom of Great Britain and Northern Ireland">United Kingdom of Great Britain and Northern Ireland</a>, citizens and residents have the freedom to change their names with relative ease.
</p><p>In theory, anyone who is at least 16 and resident in the United Kingdom can call themselves whatever they wish. However, over the past hundred years or so, formal procedures that are recognized by record holders such as government departments, companies and organizations have evolved, which enable a citizen legally to change the name recorded on their <a href="/wiki/Passport" title="Passport">passport</a>, <a href="/wiki/Driving_licence" class="mw-redirect" title="Driving licence">driving licence</a>, tax and <a href="/wiki/National_Insurance" title="National Insurance">National Insurance</a> records, bank and credit cards, etc., provided that "documentary evidence" of a change of name is provided. Documents such as <a href="/wiki/Birth_certificate" title="Birth certificate">birth</a>, <a href="/wiki/Marriage_certificate" title="Marriage certificate">marriage</a> and educational certificates cannot be changed because these documents are "matters of fact", which means that they were correct at the time they were issued.
</p><p>Documentary evidence of a change of name can be in a number of forms, such as a marriage certificate, <a href="/wiki/Decree_absolute" class="mw-redirect" title="Decree absolute">decree absolute</a>, civil partnership certificate, <a href="/wiki/Statutory_declaration" title="Statutory declaration">statutory declaration</a> or <a href="/wiki/Deed_of_change_of_name" title="Deed of change of name">deed of change of name</a>. Such documents are mere evidence that a change of name has occurred, however, and they do not themselves operate to change a person's name. Deeds of change of name are by far the most commonly used method of providing evidence of a change of name other than changing a woman's surname after marriage. A deed poll is a legal document that binds a single person to a particular course of action (in this case, changing one's name for all purposes). The term 'deed' is common to signed, written agreements that have been shown to all concerned parties. Strictly speaking, it is not a contract because it binds only one party and expresses an intention instead of a promise. 'Poll' is an old legal term referring to official documents that had cut edges (were polled) so that they were straight.<sup id="cite_ref-auto_30-0" class="reference"><a href="#cite_note-auto-30">&#91;30&#93;</a></sup><sup id="cite_ref-31" class="reference"><a href="#cite_note-31">&#91;31&#93;</a></sup>
</p>
<h3><span class="mw-headline" id="England_and_Wales">England and Wales</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=9" title="Edit section: England and Wales">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>People whose births are registered in England and Wales may have their <a href="/wiki/Deed_poll" title="Deed poll">deed poll</a> enrolled at the <a href="/wiki/Royal_Courts_of_Justice" title="Royal Courts of Justice">Royal Courts of Justice</a> in <a href="/wiki/London" title="London">London</a>.<sup id="cite_ref-32" class="reference"><a href="#cite_note-32">&#91;32&#93;</a></sup><sup id="cite_ref-auto_30-1" class="reference"><a href="#cite_note-auto-30">&#91;30&#93;</a></sup> See also <a href="/wiki/College_of_Arms#Change_of_names" title="College of Arms">College of Arms#Change of names</a>.
</p>
<h4><span class="mw-headline" id="Historical_usage">Historical usage</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=10" title="Edit section: Historical usage">edit</a><span class="mw-editsection-bracket">]</span></span></h4>
<div role="note" class="hatnote navigation-not-searchable">See also: <a href="/wiki/Double-barrelled_name#British_tradition" title="Double-barrelled name">Double-barrelled name §&#160;British tradition</a></div>
<p>From the mediæval age to the 19th century, the era of family dynasties, name changes were frequently demanded of heirs in the last wills and testaments, legacies and bequests, of members of the gentry and nobility who were the last males of their bloodline. Such persons frequently selected a younger nephew or cousin as the heir to their estates on condition that he should adopt the surname and armorials of the legator in lieu of his patronymic. Thus the ancient family otherwise destined to extinction would appear to continue as a great dynasty in the making. Such changes were also more rarely demanded by <a href="/wiki/Marriage_settlement" class="mw-redirect" title="Marriage settlement">marriage settlements</a>, for example where the father of a sole daughter and heiress demanded that as a condition of his daughter's <a href="/wiki/Dowry" title="Dowry">dowry</a> her husband should adopt his father-in-law's surname and arms. Thus the progeny of the marriage would continue the otherwise extinct family's name. Such name changes were generally only demanded of younger sons, where an elder brother was available to inherit the paternal estates under primogeniture and carry on the name and arms abandoned by the younger brother. Such name changes were effected by obtaining a private <a href="/wiki/Act_of_Parliament" title="Act of Parliament">Act of Parliament</a> or by obtaining a <a href="/wiki/College_of_Arms" title="College of Arms">Royal Licence</a>.<sup id="cite_ref-33" class="reference"><a href="#cite_note-33">&#91;33&#93;</a></sup> A less radical procedure adopted from the 18th century onwards was for the legator or settlor to demand only that the legatee or beneficiary should adopt his surname <i>in addition to</i> his patronymic, not in place of it, which gave rise to the "<a href="/wiki/Double-barrelled_name" title="Double-barrelled name">double-barrelled</a>", even the "triple-barrelled" name, frequently parodied in literature as epitomising the wealthy "<a href="/wiki/Squirearchy" class="mw-redirect" title="Squirearchy">squirearchy</a>" with an embarrassment of inherited estates.
</p><p>Well known examples are:
</p>
<ul><li>Russell to Gorges (14th century). Ralph IV Gorges, 2nd Baron Gorges, died without issue in 1331. In an effort to preserve his family name and arms he made one of his younger nephews his heir, on condition that he should adopt the name and arms of Gorges. This nephew was William Russell, the second son of his second sister Eleanor de Gorges who had married Sir Theobald Russell (d.1341) of <a href="/wiki/Kingston_Russell" title="Kingston Russell">Kingston Russell</a>, Dorset. The event is referred to in one of the earliest heraldic law cases brought concerning English armory, <a href="/wiki/Warbelton_v._Gorges" class="mw-redirect" title="Warbelton v. Gorges">Warbelton v. Gorges</a> in 1347.</li>
<li>Smithson to Percy (18th century). <a href="/wiki/Hugh_Percy,_1st_Duke_of_Northumberland" title="Hugh Percy, 1st Duke of Northumberland">Sir Hugh Smithson, 4th Baronet</a> (1715–1786) (c.1714–1786) in 1740 married <a href="/wiki/Elizabeth_Percy,_Duchess_of_Northumberland" title="Elizabeth Percy, Duchess of Northumberland">Lady Elizabeth Seymour</a>, daughter and sole heiress of <a href="/wiki/Algernon_Seymour,_7th_Duke_of_Somerset" title="Algernon Seymour, 7th Duke of Somerset">Algernon Seymour, 7th Duke of Somerset</a>, and granddaughter of Lady Elizabeth Percy (d.1722), daughter and sole heiress of <a href="/wiki/Josceline_Percy,_11th_Earl_of_Northumberland" title="Josceline Percy, 11th Earl of Northumberland">Josceline Percy, 11th Earl of Northumberland</a> (1644–1670). In 1740, by a private Act of Parliament,<sup id="cite_ref-34" class="reference"><a href="#cite_note-34">&#91;34&#93;</a></sup> Smithson changed his surname to Percy and inherited the title <a href="/wiki/Earl_of_Northumberland" title="Earl of Northumberland">Earl of Northumberland</a> and was later created <a href="/wiki/Duke_of_Northumberland" title="Duke of Northumberland">Duke of Northumberland</a>.</li></ul>
<h3><span class="mw-headline" id="Scotland">Scotland</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=11" title="Edit section: Scotland">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>People whose births are <a href="/wiki/Civil_registration" title="Civil registration">registered</a> in Scotland (or who were adopted in Scotland) can change their name in the same way as people from the rest of the United Kingdom.  However they can optionally apply to the <a href="/wiki/Registrar_General_for_Scotland" class="mw-redirect" title="Registrar General for Scotland">Registrar General for Scotland</a> to have their birth certificate amended to show the new name and have the respective register updated.
</p>
<h2><span class="mw-headline" id="Other_common_law_countries">Other common law countries</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=12" title="Edit section: Other common law countries">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Australia">Australia</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=13" title="Edit section: Australia">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Individuals may legally change their name through the <a href="/wiki/States_and_territories_of_Australia" title="States and territories of Australia">state and territory</a> governments of Australia according to state or territory laws and regulations via agencies generally titled "Registry of Births, Deaths and Marriages".<sup id="cite_ref-35" class="reference"><a href="#cite_note-35">&#91;35&#93;</a></sup>  Exceptions exist for some states such as restricted persons (e.g. jail inmate)<sup id="cite_ref-36" class="reference"><a href="#cite_note-36">&#91;36&#93;</a></sup> and in some states, changes may only be made once a year.<sup id="cite_ref-37" class="reference"><a href="#cite_note-37">&#91;37&#93;</a></sup>
</p><p>Institutions such as banks,<sup id="cite_ref-38" class="reference"><a href="#cite_note-38">&#91;38&#93;</a></sup> the Passports Office<sup id="cite_ref-39" class="reference"><a href="#cite_note-39">&#91;39&#93;</a></sup> and transport authorities<sup id="cite_ref-40" class="reference"><a href="#cite_note-40">&#91;40&#93;</a></sup> require proof of identity, so people who change their names need to have proof of the change.
</p><p>If a person's birth or adoption was registered in Australia, the change will also be noted (in most cases) on the person's birth or adoption registration, and in some states or territories, the name change can be evidenced either through a re-issued <a href="/wiki/Birth_certificate#Australia" title="Birth certificate">birth certificate</a> if born in Australia and/or "Change of Name Certificate".  These certificates are recognised secure <a href="/wiki/Identity_documents_of_Australia" title="Identity documents of Australia">identity documents</a> and can be verified electronically through the <a href="/wiki/Attorney-general_of_Australia" class="mw-redirect" title="Attorney-general of Australia">Attorney-general of Australia</a>'s Document Verification Service.<sup id="cite_ref-41" class="reference"><a href="#cite_note-41">&#91;41&#93;</a></sup>
</p>

<h3><span class="mw-headline" id="Canada">Canada</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=14" title="Edit section: Canada">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In Canada a person can informally call themselves whatever they want. In all provinces except <a href="/wiki/Quebec" title="Quebec">Quebec</a>,<sup id="cite_ref-42" class="reference"><a href="#cite_note-42">&#91;42&#93;</a></sup> when someone gets married they can change their last name without legally changing their name by using their <a href="/wiki/Marriage_Certificate" class="mw-redirect" title="Marriage Certificate">Marriage Certificate</a> as verification of the name change.<sup id="cite_ref-43" class="reference"><a href="#cite_note-43">&#91;43&#93;</a></sup> It is almost impossible for married women in Québec to take their husbands' names.<sup id="cite_ref-44" class="reference"><a href="#cite_note-44">&#91;44&#93;</a></sup>
Like the United States, the requirements for a formal name change varies from province to province. Except for <a href="/wiki/Ontario" title="Ontario">Ontario</a>, <a href="/wiki/British_Columbia" title="British Columbia">British Columbia</a> and <a href="/wiki/New_Brunswick" title="New Brunswick">New Brunswick</a>, <a href="/wiki/Canadians" title="Canadians">Canadians</a> must be 18 to change their names and have lived in the province they are changing it in for at least 3 months to a year, depending on province. (In <a href="/wiki/British_Columbia" title="British Columbia">British Columbia</a>, Canadians must be 19 to change their names, in <a href="/wiki/Ontario" title="Ontario">Ontario</a> and <a href="/wiki/New_Brunswick" title="New Brunswick">New Brunswick</a> 16.) People younger than the province's <a href="/wiki/Age_of_majority" title="Age of majority">age of majority</a> can change their names if they have their guardians' consent, are legally married, or have a <a href="/wiki/Common_law_marriage" class="mw-redirect" title="Common law marriage">common law marriage</a>.<sup id="cite_ref-cbc.ca_45-0" class="reference"><a href="#cite_note-cbc.ca-45">&#91;45&#93;</a></sup> To change a name, a Name Change application must be submitted to either the Ministry of Government Services, Court of justice or Registrar of civil status. A document such as a birth certificate must be submitted. A statement as to why the name is being changed is needed in most areas and the reason has to be serious. In Canada, a name cannot cause confusion, be used for misrepresentation or fraud and in most cases the name change is announced in newspapers. There is a fee involved and it ranges from $10 to $185 depending on the territory or province.<sup id="cite_ref-cbc.ca_45-1" class="reference"><a href="#cite_note-cbc.ca-45">&#91;45&#93;</a></sup> <a href="/wiki/British_Columbia" title="British Columbia">British Columbia</a>, <a href="/wiki/Alberta" title="Alberta">Alberta</a> and <a href="/wiki/Manitoba" title="Manitoba">Manitoba</a> need <a href="/wiki/Fingerprint_recognition" class="mw-redirect" title="Fingerprint recognition">fingerprints</a> to be submitted before one changes their name.<sup id="cite_ref-46" class="reference"><a href="#cite_note-46">&#91;46&#93;</a></sup><sup id="cite_ref-47" class="reference"><a href="#cite_note-47">&#91;47&#93;</a></sup>
<sup id="cite_ref-48" class="reference"><a href="#cite_note-48">&#91;48&#93;</a></sup>
</p>
<h3><span class="mw-headline" id="Hong_Kong">Hong Kong</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=15" title="Edit section: Hong Kong">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<table class="box-Unreferenced_section plainlinks metadata ambox ambox-content ambox-Unreferenced" role="presentation"><tbody><tr><td class="mbox-image"><div style="width:52px"><a href="/wiki/File:Question_book-new.svg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/en/thumb/9/99/Question_book-new.svg/50px-Question_book-new.svg.png" decoding="async" width="50" height="39" srcset="//upload.wikimedia.org/wikipedia/en/thumb/9/99/Question_book-new.svg/75px-Question_book-new.svg.png 1.5x, //upload.wikimedia.org/wikipedia/en/thumb/9/99/Question_book-new.svg/100px-Question_book-new.svg.png 2x" data-file-width="512" data-file-height="399" /></a></div></td><td class="mbox-text"><div class="mbox-text-span">This section <b>does not <a href="/wiki/Wikipedia:Citing_sources" title="Wikipedia:Citing sources">cite</a> any <a href="/wiki/Wikipedia:Verifiability" title="Wikipedia:Verifiability">sources</a></b>.<span class="hide-when-compact"> Please help <a class="external text" href="https://en.wikipedia.org/w/index.php?title=Name_change&amp;action=edit">improve this section</a> by <a href="/wiki/Help:Introduction_to_referencing_with_Wiki_Markup/1" title="Help:Introduction to referencing with Wiki Markup/1">adding citations to reliable sources</a>. Unsourced material may be challenged and <a href="/wiki/Wikipedia:Verifiability#Burden_of_evidence" title="Wikipedia:Verifiability">removed</a>.</span>  <small class="date-container"><i>(<span class="date">June 2010</span>)</i></small><small class="hide-when-compact"><i> (<a href="/wiki/Help:Maintenance_template_removal" title="Help:Maintenance template removal">Learn how and when to remove this template message</a>)</i></small></div></td></tr></tbody></table>
<p>It is a common practice for ethnic Chinese residents of Hong Kong to adopt a western-style English name in addition to their transliterated Chinese name. As they often adopt western-style English names after being registered on the birth register, the fact that they want to include a western-style English name as part of their <a href="/wiki/Legal_name" title="Legal name">legal English name</a> is regarded as a name change which usually requires a deed poll.
</p><p>However, the <a href="/wiki/Immigration_Department_(Hong_Kong)" title="Immigration Department (Hong Kong)">Immigration Department</a> which is responsible for processing applications for name change allows applicants to submit such applications without deeds poll; anyone who has a phonetic English name only and wishes to include a western-style English name as part of his or her legal English name can apply to the Immigration Department without a deed poll. Only one application of this kind is allowed for each applicant; any application for subsequent change(s) must be made with a deed poll.
</p>
<h3><span class="mw-headline" id="Ireland">Ireland</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=16" title="Edit section: Ireland">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In the <a href="/wiki/Republic_of_Ireland" title="Republic of Ireland">Republic of Ireland</a>, a person earns their name by "use and repute".<sup id="cite_ref-citizensinformation_49-0" class="reference"><a href="#cite_note-citizensinformation-49">&#91;49&#93;</a></sup> For most purposes it is enough to simply use the desired name and ask others to call you by that name.<sup id="cite_ref-citizensinformation_49-1" class="reference"><a href="#cite_note-citizensinformation-49">&#91;49&#93;</a></sup> This was the traditional practice for a bride adopting her husband's surname.<sup id="cite_ref-citizensinformation_49-2" class="reference"><a href="#cite_note-citizensinformation-49">&#91;49&#93;</a></sup> A <a href="/wiki/Deed_of_change_of_name" title="Deed of change of name">change of name deed poll</a> is not required, but provides documentary evidence of a name change.<sup id="cite_ref-citizensinformation_49-3" class="reference"><a href="#cite_note-citizensinformation-49">&#91;49&#93;</a></sup> Resident non-<a href="/wiki/Citizenship_of_the_European_Union" title="Citizenship of the European Union">EU nationals</a> must apply to the <a href="/wiki/Irish_Naturalisation_and_Immigration_Service" title="Irish Naturalisation and Immigration Service">Irish Naturalisation and Immigration Service</a> for a deed poll.<sup id="cite_ref-citizensinformation_49-4" class="reference"><a href="#cite_note-citizensinformation-49">&#91;49&#93;</a></sup> The deed poll requires a witness <a href="/wiki/Affidavit" title="Affidavit">affidavit</a>, and may optionally be enrolled in the <a href="/wiki/High_Court_(Ireland)" title="High Court (Ireland)">High Court</a> upon payment of <a href="/wiki/Stamp_duty" title="Stamp duty">stamp duty</a>.<sup id="cite_ref-citizensinformation_49-5" class="reference"><a href="#cite_note-citizensinformation-49">&#91;49&#93;</a></sup> An enrolled deed poll is required for some administrative name changes, such as on a <a href="/wiki/Driving_licence_in_the_Republic_of_Ireland" title="Driving licence in the Republic of Ireland">driving licence</a><sup id="cite_ref-citizensinformation_49-6" class="reference"><a href="#cite_note-citizensinformation-49">&#91;49&#93;</a></sup> or when <a href="/wiki/Transgender_rights_in_Ireland" title="Transgender rights in Ireland">changing legal gender</a>.<sup id="cite_ref-50" class="reference"><a href="#cite_note-50">&#91;50&#93;</a></sup>
</p>
<h3><span class="mw-headline" id="New_Zealand">New Zealand</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=17" title="Edit section: New Zealand">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>From September 1995, <a href="/wiki/New_Zealanders" title="New Zealanders">New Zealanders</a> can change their name by making a <a href="/wiki/Statutory_declaration" title="Statutory declaration">statutory declaration</a> and, if approved, the new name is registered with the Births, Deaths and Marriages section of the Department of Internal Affairs (Identity Services). Prior to September 1995, they changed their name by <a href="/wiki/Deed_poll" title="Deed poll">deed poll</a>.<sup id="cite_ref-51" class="reference"><a href="#cite_note-51">&#91;51&#93;</a></sup>
</p>
<h2><span class="mw-headline" id="Civil_law">Civil law</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=18" title="Edit section: Civil law">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In general, unlike in <a href="/wiki/Common_law" title="Common law">common law</a> countries, names cannot be changed at will in <a href="/wiki/Civil_law_(legal_system)" title="Civil law (legal system)">civil law</a> jurisdictions. Usually, a name change requires government approval and is only rarely granted, though legal name changes have become more common in some jurisdictions over the last years. The reason given for this system is usually the public interest in the unique identifiability of a person, e.g., in governmental registers, although with the advent of <a href="/wiki/Personal_identification_numbers" class="mw-redirect" title="Personal identification numbers">personal identification numbers</a>, that rationale may be in need of reconsideration.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">&#91;<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (March 2011)">citation needed</span></a></i>&#93;</sup>
</p>
<h3><span class="mw-headline" id="Belgium">Belgium</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=19" title="Edit section: Belgium">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Changes in law Aug 2018 changes first name and fee will fall under the authority of local town hall <sup id="cite_ref-52" class="reference"><a href="#cite_note-52">&#91;52&#93;</a></sup> In Belgian law, a name is in principle considered fixed for life, but under exceptional circumstances, a person may apply to the Ministry of Justice for a name change. This requires a Royal Decree (French: <i>Arrêté royal</i>, Dutch: <i>Koninklijk besluit</i>) for last names, but only a Ministerial Decree for first names. The new name must not cause confusion or cause damage to the bearer or others. Examples of requests that are usually considered favorably:
</p>
<ul><li>A person of non-European origin who wants to adopt a less exotic name to further his/her integration in Belgian society</li>
<li>A person stuck with a ridiculous last name that is causing him/her great embarrassment or emotional distress. Actual examples: Salami, Naaktgeboren ("born naked"), and Clooten ("sods of earth" in Middle Dutch, but "testicles" in modern Dutch)</li>
<li>(for minor children) following legal adoption or recognition of paternity<sup id="cite_ref-53" class="reference"><a href="#cite_note-53">&#91;53&#93;</a></sup></li></ul>
<h3><span class="mw-headline" id="Brazil">Brazil</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=20" title="Edit section: Brazil">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>According to the Brazilian Civil Code and the Public Registries Act, the name registered after birth is definitive and immutable. However, there are some circumstances under which a name change is allowed: 
</p>
<ul><li>If an obvious writing error is made while registering the name of the child.</li>
<li>In the first year after a person reaches the <a href="/wiki/Age_of_majority" title="Age of majority">age of majority</a> (18&#160;years), they can request a change in their first names, but modification of surnames is not allowed.</li>
<li>If a person's name exposes them to ridicule and harms their everyday well-being.</li>
<li>If a person is recognized publicly by another name, they can request a name change or addition, by providing as support three testimonials of the change in name.</li>
<li>If a person has undergone a sex change, recent jurisprudence has allowed name modifications in most such cases.</li></ul>
<h3><span class="mw-headline" id="India">India</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=21" title="Edit section: India">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In India, the person concerned submits a name change request to an appropriate authority, with supporting documents. Subsequently, an application must be made to the Government Printing Press, which issues an Official Gazette Notification certifying the change of name.<sup id="cite_ref-54" class="reference"><a href="#cite_note-54">&#91;54&#93;</a></sup>
</p>
<h3><span class="mw-headline" id="Norway">Norway</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=22" title="Edit section: Norway">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<table class="box-Unreferenced_section plainlinks metadata ambox ambox-content ambox-Unreferenced" role="presentation"><tbody><tr><td class="mbox-image"><div style="width:52px"><a href="/wiki/File:Question_book-new.svg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/en/thumb/9/99/Question_book-new.svg/50px-Question_book-new.svg.png" decoding="async" width="50" height="39" srcset="//upload.wikimedia.org/wikipedia/en/thumb/9/99/Question_book-new.svg/75px-Question_book-new.svg.png 1.5x, //upload.wikimedia.org/wikipedia/en/thumb/9/99/Question_book-new.svg/100px-Question_book-new.svg.png 2x" data-file-width="512" data-file-height="399" /></a></div></td><td class="mbox-text"><div class="mbox-text-span">This section <b>does not <a href="/wiki/Wikipedia:Citing_sources" title="Wikipedia:Citing sources">cite</a> any <a href="/wiki/Wikipedia:Verifiability" title="Wikipedia:Verifiability">sources</a></b>.<span class="hide-when-compact"> Please help <a class="external text" href="https://en.wikipedia.org/w/index.php?title=Name_change&amp;action=edit">improve this section</a> by <a href="/wiki/Help:Introduction_to_referencing_with_Wiki_Markup/1" title="Help:Introduction to referencing with Wiki Markup/1">adding citations to reliable sources</a>. Unsourced material may be challenged and <a href="/wiki/Wikipedia:Verifiability#Burden_of_evidence" title="Wikipedia:Verifiability">removed</a>.</span>  <small class="date-container"><i>(<span class="date">December 2012</span>)</i></small><small class="hide-when-compact"><i> (<a href="/wiki/Help:Maintenance_template_removal" title="Help:Maintenance template removal">Learn how and when to remove this template message</a>)</i></small></div></td></tr></tbody></table>
<p>Although it has always been relatively easy to change one's legal names in Norway, it used to require some kind of government approval. As late as 1830, local vicars were instructed to write both given (Christian) names, as well as last names, in the baptismal record. Earlier, only the given name of the child, birth date, baptismal date, and sex were written down, alongside the parents' names. It was not until the beginning of the 20th century, however, that the authorities required everyone to adopt a family surname. Until about 1980, the government still required that a name change applicant apply to the government regional representative (fylkesmann). The law has been replaced twice since then. Nowadays, the process is as easy as in common-law countries; the subject merely submits the names wanted (providing that the surname chosen is not in use or is not used by fewer than 200 persons) to the local authorities for the purposes of election rosters and census counts; there is no longer an application process.
</p>
<h3><span class="mw-headline" id="Philippines">Philippines</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=23" title="Edit section: Philippines">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>2001 RA 9048 amends Articles 376 and 412 of the <a href="/wiki/Civil_Code_of_the_Philippines" title="Civil Code of the Philippines">Civil Code of the Philippines</a>, which prohibit the change of name or surname of a person, or any correction or change of entry in a civil register without a judicial order. Administrative Order No. 1 Series of 2001, implemented the law.<sup id="cite_ref-55" class="reference"><a href="#cite_note-55">&#91;55&#93;</a></sup> It authorizes the city or municipal civil registrar or the consul general to correct a clerical or typographical error in an entry or change the first name or nickname in the civil register without need of a judicial order.<sup id="cite_ref-56" class="reference"><a href="#cite_note-56">&#91;56&#93;</a></sup><sup id="cite_ref-57" class="reference"><a href="#cite_note-57">&#91;57&#93;</a></sup>
</p>
<h4><span class="mw-headline" id="Name_and_gender_change">Name and gender change</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=24" title="Edit section: Name and gender change">edit</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>In case of controversial and substantial changes, <a href="/wiki/Philippines" title="Philippines">Philippines</a> <a href="/wiki/Jurisprudence" title="Jurisprudence">jurisprudence</a> requires full-blown court <a href="/wiki/Lawsuit" title="Lawsuit">lawsuit</a>, that must include the local civil registrar in the petition, since RA 9048 and Rule 108 (Cancellation or correction of entries in the Civil Registry) of the Rules of Court do not allow the change of sex in a birth certificate.<sup id="cite_ref-58" class="reference"><a href="#cite_note-58">&#91;58&#93;</a></sup>
</p><p>The only landmark case in the Philippines on name and legal sex change is the Jeff Cagandahan case. The <a href="/wiki/Supreme_Court_of_the_Philippines" title="Supreme Court of the Philippines">Supreme Court of the Philippines</a> Justice <a href="/wiki/Leonardo_Quisumbing" title="Leonardo Quisumbing">Leonardo Quisumbing</a> on September 12, 2008, allowed Cagandahan, 27, who has <a href="/wiki/Congenital_adrenal_hyperplasia" title="Congenital adrenal hyperplasia">congenital adrenal hyperplasia</a>, to change his <a href="/wiki/Birth_certificate" title="Birth certificate">birth certificate</a> name from Jennifer to Jeff, and his legal gender from female to male.<sup id="cite_ref-59" class="reference"><a href="#cite_note-59">&#91;59&#93;</a></sup><sup id="cite_ref-60" class="reference"><a href="#cite_note-60">&#91;60&#93;</a></sup><sup id="cite_ref-61" class="reference"><a href="#cite_note-61">&#91;61&#93;</a></sup>
</p>
<h3><span class="mw-headline" id="Quebec">Quebec</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=25" title="Edit section: Quebec">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Although as in other jurisdictions residents of <a href="/wiki/Quebec" title="Quebec">Quebec</a> may informally use whatever name they want, procedures for formal name change are very strict as Quebec (unlike the rest of Canada) operates under a <a href="/wiki/Civil_law_(legal_system)" title="Civil law (legal system)">civil law</a> system. The decision must be authorized by the Director of Civil Status, and requires a valid reason for changing the name, including long-term use of the new name (in the Montreuil case cited below, the Quebec appeals court has considered five years' use to be a sufficient reason), difficulty of use due to spelling or pronunciation, or bearing a name that another person has made infamous.
</p><p>Only a judge may authorize a name change for a child for reasons of abandonment, deprivation of parental authority, or change in <a href="/wiki/Paternity_(law)" class="mw-redirect" title="Paternity (law)">filiation</a> such as <a href="/wiki/Adoption" title="Adoption">adoption</a>.
</p><p>This has occasionally led to controversy. A lawyer named Micheline Montreuil, a non-operative <a href="/wiki/Transgender" title="Transgender">transgender</a> woman, had to undergo a lengthy process to have her name legally changed. Initially, the director of civil status refused to permit the change on the grounds that a legal male could not bear a female name. According to Quebec law, Montreuil could not change her record of sex because this requires proof of a completed <a href="/wiki/Sex_reassignment_surgery" title="Sex reassignment surgery">sex reassignment surgery</a>, which she has not had. On November 1, 1999, the provincial court of appeal ruled that nothing in the law prevented a person who was legally male from legally adopting a woman's name. (Montreuil was initially prevented from changing her name despite this ruling on the grounds that she had not established general use, as normally required for a name change; the Quebec appeals court finally authorized the change on November 7, 2002.)<sup id="cite_ref-62" class="reference"><a href="#cite_note-62">&#91;62&#93;</a></sup>
</p><p>The Director of Civil Status will amend a Quebec birth certificate if a name change certificate is issued by another province. Some have used that loophole by temporarily moving to one of Canada's other provinces, which follow the more permissive common-law rules, in order to get the legal documents.
</p>
<h3><span class="mw-headline" id="South_Africa">South Africa</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=26" title="Edit section: South Africa">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>South Africa, which uses a mixture of common law and civil <a href="/wiki/Roman_Dutch_law" class="mw-redirect" title="Roman Dutch law">Roman Dutch law</a>, mostly uses common-law procedures with regard to name change. Name changes in <a href="/wiki/South_Africa" title="South Africa">South Africa</a> are regulated by the Births and Deaths Registration Act (Act 51 of 1992, as amended). The personal information of all citizens and permanent residents is recorded on the Population Register, so any name changes must be registered.
</p><p>A person can change their forenames by submitting a form to the <a href="/wiki/South_African_Department_of_Home_Affairs" class="mw-redirect" title="South African Department of Home Affairs">Department of Home Affairs</a>. An individual's surname, or that of a family, may be changed by applying to the Department and providing a "good and sufficient reason" for the change.
</p><p>A married woman can change her surname to that of her husband or join her maiden name with her husband's surname, and a divorced woman may return to her previous surname, without applying or paying a fee; but she must notify the department so that the details in the Population Register can be changed. (It is possible that, if challenged, these provisions might be held to be unconstitutional because they apply only to women.)
</p><p>South Africa has officially recognized same sex marriages since 2006 and in doing so now allows one or both partners to change their surnames in the marriage register on the day of the marriage. A new passport and ID book can then be applied for with the new married surname as well.
</p><p>The surnames of minor children can also be changed under various circumstances involving the marriage, divorce or death of a parent, children born out of wedlock, and guardianship.<sup id="cite_ref-63" class="reference"><a href="#cite_note-63">&#91;63&#93;</a></sup>
</p>
<h3><span class="mw-headline" id="Switzerland">Switzerland</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=27" title="Edit section: Switzerland">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In Switzerland, a name change requires the approval of the respective <a href="/wiki/Cantons_of_Switzerland" title="Cantons of Switzerland">Cantonal</a> government, if there are important reasons (<i>wichtige Gründe</i> / <i>justes motifs</i>) for the change, according to article 30 of the <a href="/wiki/Swiss_Civil_Code" title="Swiss Civil Code">Swiss Civil Code</a>. When aliens apply for naturalization, they have the option of asking for their names to be changed upon the grants of citizenship with no additional fees.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">&#91;<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (February 2013)">citation needed</span></a></i>&#93;</sup> According to the case law of the <a href="/wiki/Swiss_Federal_Supreme_Court" class="mw-redirect" title="Swiss Federal Supreme Court">Swiss Federal Supreme Court</a>, such requests must be granted only if the petitioner shows that they suffer substantially from their present name, e.g., if it is the same as that of a notorious criminal.
</p>
<h3><span class="mw-headline" id="Taiwan">Taiwan</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=28" title="Edit section: Taiwan">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<table role="presentation" class="mbox-small plainlinks sistersitebox" style="background-color:#f9f9f9;border:1px solid #aaa;color:#000">
<tbody><tr>
<td class="mbox-image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/df/Wikibooks-logo-en-noslogan.svg/40px-Wikibooks-logo-en-noslogan.svg.png" decoding="async" width="40" height="40" class="noviewer" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/df/Wikibooks-logo-en-noslogan.svg/60px-Wikibooks-logo-en-noslogan.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/df/Wikibooks-logo-en-noslogan.svg/80px-Wikibooks-logo-en-noslogan.svg.png 2x" data-file-width="400" data-file-height="400" /></td>
<td class="mbox-text plainlist">Wikibooks has a book on the topic of: <i><b><a href="https://en.wikibooks.org/wiki/Annotated_Republic_of_China_Laws/Name_Act" class="extiw" title="wikibooks:Annotated Republic of China Laws/Name Act">Annotated Republic of China Laws/Name Act</a></b></i></td></tr>
</tbody></table>
<p>In <a href="/wiki/Taiwan" title="Taiwan">Taiwan</a>, the government strictly regulates when one's surname, given name or both may or may not be changed, under the Name Act (姓名條例) since 1953 with 10 articles, totally amended into 14 articles in 2001, and the <a href="https://en.wikisource.org/wiki/Enforcement_Regulations_of_the_Name_Act" class="extiw" title="s:Enforcement Regulations of the Name Act">Enforcement Regulations of the Name Act</a> (姓名條例施行細則).
</p><p>Legally permitted reasons to change one's given name are:<sup id="cite_ref-64" class="reference"><a href="#cite_note-64">&#91;64&#93;</a></sup>
</p>
<ol><li>Having the same name when serving in an institution or studying in the same school.</li>
<li>Since 1983, having the same name as one's parent, grandparent, or great-grandparent.<sup id="cite_ref-65" class="reference"><a href="#cite_note-65">&#91;65&#93;</a></sup></li>
<li>Having the same name as someone else who lives in the same county or city and having lived there for at least 6 months.</li>
<li>Having the same name as a wanted criminal, for example <a href="https://zh.wikipedia.org/wiki/%E9%99%B3%E9%80%B2%E8%88%88" class="extiw" title="zh:陳進興">陳進興 (Chén Jìnxīng)</a>, a common name used by not only a convicted kidnapper and murderer who was executed in Taiwan for major crimes in 1997 (<a href="https://zh.wikipedia.org/wiki/%E9%99%B3%E9%80%B2%E8%88%88_(%E5%8F%B0%E7%81%A3%E7%BD%AA%E7%8A%AF)" class="extiw" title="zh:陳進興 (台灣罪犯)">zh:陳進興 (台灣罪犯)</a>) but also a Taiwanese statesman (<a href="https://zh.wikipedia.org/wiki/%E9%99%B3%E9%80%B2%E8%88%88_(%E5%8F%B0%E7%81%A3%E6%94%BF%E6%B2%BB%E4%BA%BA%E7%89%A9)" class="extiw" title="zh:陳進興 (台灣政治人物)">zh:陳進興 (台灣政治人物)</a>).</li>
<li>Having special reasons,<sup id="cite_ref-66" class="reference"><a href="#cite_note-66">&#91;66&#93;</a></sup> which is limited to twice in one's lifetime, while the second given name change of this type may not be done before reaching adulthood.<sup id="cite_ref-67" class="reference"><a href="#cite_note-67">&#91;67&#93;</a></sup></li></ol>
<h2><span class="mw-headline" id="Name_change_on_religious_conversion">Name change on religious conversion</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=29" title="Edit section: Name change on religious conversion">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div role="note" class="hatnote navigation-not-searchable">Further information: <a href="/wiki/Religious_name" title="Religious name">Religious name</a></div>
<p>Adherents of various religions change their name upon <a href="/wiki/Religious_conversion" title="Religious conversion">conversion</a> or confirmation. The name adopted may not have any legal status but will represent their adopted religious beliefs.<sup id="cite_ref-68" class="reference"><a href="#cite_note-68">&#91;68&#93;</a></sup>
</p>
<h3><span class="mw-headline" id="Buddhism">Buddhism</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=30" title="Edit section: Buddhism">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul><li>Individuals who attend a ceremony to officially become Buddhists are usually given a new "<a href="/wiki/Dharma_name" title="Dharma name">Dharma name</a>", which marks their "<a href="/wiki/Refuge_(Buddhism)" title="Refuge (Buddhism)">taking refuge</a>".</li></ul>
<h3><span class="mw-headline" id="Christianity">Christianity</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=31" title="Edit section: Christianity">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul><li>It has been historical Christian practice to adopt a name on <a href="/wiki/Baptism" title="Baptism">baptism</a> or (in some countries) <a href="/wiki/Confirmation_(sacrament)" class="mw-redirect" title="Confirmation (sacrament)">confirmation</a>.<sup id="cite_ref-69" class="reference"><a href="#cite_note-69">&#91;69&#93;</a></sup></li>
<li>The customary practice whereby persons entering a <a href="/wiki/Religious_institute" title="Religious institute">religious institute</a> take on a <a href="/wiki/Name_in_religion" class="mw-redirect" title="Name in religion">name in religion</a> is still observed by <a href="/wiki/Eastern_Orthodox" class="mw-redirect" title="Eastern Orthodox">Eastern Orthodox</a> and some traditional <a href="/wiki/Roman_Catholic_Church" class="mw-redirect" title="Roman Catholic Church">Roman Catholics</a>. Eastern Orthodox monastics are usually given the name of a prophet or a monastic saint.</li>
<li><a href="/wiki/Pope" title="Pope">Popes</a> take a <a href="/wiki/Papal_name" title="Papal name">papal name</a> upon their accession to office; for example, <a href="/wiki/Jorge_M._Bergoglio" class="mw-redirect" title="Jorge M. Bergoglio">Jorge M. Bergoglio</a> adopted the name <a href="/wiki/Francis_(given_name)" title="Francis (given name)">Francis</a>.</li></ul>
<h3><span class="mw-headline" id="Hinduism">Hinduism</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=32" title="Edit section: Hinduism">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>There is no formal concept of conversion in Hinduism but converts to Hinduism are accepted, usually after a small ceremony called <a href="/wiki/Shuddhi" title="Shuddhi">Shudhikaran</a> (purification). Individuals who attend a Shudhikaran ceremony to officially become <a href="/wiki/Hindu" class="mw-redirect" title="Hindu">Hindu</a> may be optionally given a new <a href="/wiki/Dharma" title="Dharma">Dharma</a> (religious) name, which is usually based on <a href="/wiki/Sanskrit" title="Sanskrit">Sanskrit</a> or <a href="/wiki/Indian_name" title="Indian name">Indian name</a> such as names based on <a href="/wiki/Hindu_deities" title="Hindu deities">Hindu deities</a>.
</p>
<h3><span class="mw-headline" id="Islam">Islam</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=33" title="Edit section: Islam">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul><li>Converts to Islamic faith may choose a new name; although it is not required, it may in some cases be preferable for personal reasons or because the name is of uncertain Islamicity (e.g. the person converting has a <a href="/wiki/Theophoric_name" title="Theophoric name">theophoric name</a> in another religion, such as Christopher or Ganesh). Boxer Cassius Clay's adoption of the name <a href="/wiki/Muhammad_Ali" title="Muhammad Ali">Muhammad Ali</a> is a well-known example, as is Cat Stevens' change to <a href="/wiki/Yusuf_Islam" class="mw-redirect" title="Yusuf Islam">Yusuf Islam</a> and Malcolm Little's adoption of the name <a href="/wiki/Malcolm_X" title="Malcolm X">Malcolm X</a> and later <i>El-Hajj Malik El-Shabazz</i>.  On the other hand, converts may choose to keep their names, as did <a href="/wiki/Dave_Chappelle" title="Dave Chappelle">Dave Chappelle</a>.</li>
<li>Women do not normally take their husbands' surnames as their own. Their maiden name continues to be their surname even after marriage. Sometimes their husbands first name becomes their surname, similar to the case of surname chains indicating someone's chain of fathers.</li>
<li>In Islamic tradition, there are two kinds of <a href="/wiki/Surname" title="Surname">surname</a>.</li></ul>
<dl><dd><ol><li>The first kind comes from the father's first name, which means that the particular person is the child of. For example, a name such as <i>ibn Abdullah</i> means "son of Abdullah", or <i>bint Abdullah</i> means "daughter of Abdullah". Father descendant names can be chained, meaning that an individual can have the name of their father, followed by their paternal grandfather, followed by the father of that grandfather, to indicate a patrilineal lineage.</li>
<li>The second kind is associated with <a href="/wiki/Clan" title="Clan">clan</a>, <a href="/wiki/Tribal" class="mw-redirect" title="Tribal">tribal</a> or <a href="/wiki/Ethnic" class="mw-redirect" title="Ethnic">ethnic</a> affiliation and is also solely dependant on the person's patrilineal line. In Islamic family law, the individual is required to keep the clan, tribal or ethnic affiliation of their father where it is known, whether the child be illegitimate, staying with their mother after divorce or adopted.</li></ol></dd></dl>
<p>The last name cannot be changed in such a way as to reflect a heritage that is not that of one's biological father.
</p>
<h3><span class="mw-headline" id="Judaism">Judaism</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=34" title="Edit section: Judaism">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul><li>Jewish people in the Diaspora sometimes give their children two names: a secular name for everyday use and a Hebrew name for religious purposes.  <a href="/wiki/Conversion_to_Judaism" title="Conversion to Judaism">Converts to Judaism</a> choose a Hebrew name.<sup id="cite_ref-70" class="reference"><a href="#cite_note-70">&#91;70&#93;</a></sup> Full Jewish names include a <a href="/wiki/Patronym" class="mw-redirect" title="Patronym">patronym</a>: converts take the patronym "ben/bat Avraham Avinu" (son/daughter of Our Father Abraham) as converts are held to be spiritual descendants of <a href="/wiki/Abraham" title="Abraham">Abraham</a>, the forebear of Jews.</li></ul>
<h3><span class="mw-headline" id="Sikhism">Sikhism</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=35" title="Edit section: Sikhism">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul><li>Those in the <a href="/wiki/Sikh" class="mw-redirect" title="Sikh">Sikh</a> faith adopt a new last name upon baptism into the <a href="/wiki/Khalsa" title="Khalsa">Khalsa</a>. Men adopt the last name <a href="/wiki/Singh" title="Singh">Singh</a>, while women adopt the last name <a href="/wiki/Kaur" title="Kaur">Kaur</a>.</li>
<li>The Sikhs adopted the name Singh in 1699 during the Birth of the Khalsa.</li></ul>
<h3><span class="mw-headline" id="Wicca">Wicca</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Name_change&amp;action=edit&amp;section=36" title="Edit section: Wicca">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul><li>Some <a href="/wiki/Wiccans" class="mw-redirect" title="Wiccans">Wiccans</a> have a <a href="/wiki/Craft_name" title="Craft name">craft name</a> that is used during coven gatherings and rituals.</li></ul>

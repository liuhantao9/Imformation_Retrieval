	<h1 id="firstHeading" class="firstHeading" lang="en">Javanese name</h1>

<p><b><a href="/wiki/Javanese_people" title="Javanese people">Javanese people</a></b> have various systems for <a href="/wiki/Name" title="Name">naming</a>. Some Javanese, especially those from older generations, have <a href="/wiki/Mononymous_person" title="Mononymous person">only one name and no surname</a>. Others use their father's names as well as their own, in a similar manner to European <a href="/wiki/Patronymic" title="Patronymic">patronymics</a>. For example, <a href="/wiki/Abdurrahman_Wahid" title="Abdurrahman Wahid">Abdurrahman Wahid</a>'s name is derived from <a href="/wiki/Wahid_Hasyim" title="Wahid Hasyim">Wahid Hasyim</a>, his father, an independence fighter and minister.  In turn, Wahid Hasyim's name was derived from his father named <a href="/wiki/Hasyim_Asyari" class="mw-redirect" title="Hasyim Asyari">Hasyim Asyari</a>, a cleric and founder of the <a href="/wiki/Nahdlatul_Ulama" title="Nahdlatul Ulama">Nahdlatul Ulama</a> organization. Another example is former <a href="/wiki/President_of_Indonesia" title="President of Indonesia">president</a> <a href="/wiki/Megawati_Sukarnoputri" title="Megawati Sukarnoputri">Megawati Sukarnoputri</a>; the last part of the name is a patronymic, meaning "<a href="/wiki/Sukarno" title="Sukarno">Sukarno's</a> daughter".
</p><p>Culturally, Javanese people use a <a href="/wiki/Patrilineality" title="Patrilineality">patrilineal</a> system that traces the hierarchic lineage of the father. This system is particularly used to determine descendants' rights to use royal titles before their names. However, it is not customary for Javanese to pass on a <a href="/wiki/Family_name" class="mw-redirect" title="Family name">family name</a>, except in <a href="/wiki/Suriname" title="Suriname">Suriname</a>, which has a large Javanese population. Surnames in Suriname Javanese are usually derived from the names of their ancestors who emigrated from <a href="/wiki/Java_(island)" class="mw-redirect" title="Java (island)">Java</a> between 1890-1939.  Suriname Javanese people usually used Western (mostly Dutch) given names, and Javanese surnames, many of which are archaic in Java itself. The examples of Suriname Javanese surnames are Atmodikoro, Bandjar, Dasai, Hardjoprajitno, Irodikromo, Kromowidjojo, Moestadja, Pawironadi, Redjosentono, Somohardjo, etc. Other Javanese communities who have surnames are the <a href="/w/index.php?title=Jatons&amp;action=edit&amp;redlink=1" class="new" title="Jatons (page does not exist)">Jatons</a> (Jawa Tondano/Tondano Javanese), descendants of Prince Diponegoro's followers exiled to <a href="/wiki/North_Sulawesi" title="North Sulawesi">North Sulawesi</a>. Some of their surnames are Arbi, Baderan, Djoyosuroto, Guret, Kiaidemak, Modjo, Ngurawan, Pulukadang, Suratinoyo, Wonopati, Zees, etc.
</p><p>Some Javanese, especially those from older generations, have a single name, for example, <a href="/wiki/Sukarno" title="Sukarno">Sukarno</a>, <a href="/wiki/Suharto" title="Suharto">Suharto</a>, and <a href="/wiki/Boediono" title="Boediono">Boediono</a>. Some names are derived from native <a href="/wiki/Javanese_language" title="Javanese language">Javanese language</a>, while some others are derived from <a href="/wiki/Sanskrit" title="Sanskrit">Sanskrit</a>. Names with the prefix <i>Su-</i>, which means <i>good</i>, are very popular. After the advent of <a href="/wiki/Islam" title="Islam">Islam</a>, many Javanese used <a href="/wiki/Arabic_name" title="Arabic name">Arabic names</a>, especially those amongst clerics and the northern coast population, where Islamic influence is stronger. There are many Javanese-style <a href="/wiki/Arabic_name" title="Arabic name">Arabic names</a> such as Marpuah (from Marfu'ah), Ngabdurohman  (from <a href="/wiki/Abdurrahman" class="mw-redirect" title="Abdurrahman">Abdurrahman</a>), Sarip (from Sharif), Slamet (from Salamah), Solikin (from Salihin), etc.
</p><p>In the past, commoners usually had only one-word names, while nobles had names of two or more words but rarely used a surname. Due to the influence of other cultures, many started using names from other languages, mainly <a href="/wiki/European_languages" class="mw-redirect" title="European languages">European</a>. Catholic Javanese usually use <a href="/wiki/Latin" title="Latin">Latin</a> <a href="/wiki/Given_name" title="Given name">baptismal names</a> followed by a traditional Javanese name, for example <a href="/wiki/Albertus_Soegijopranoto" class="mw-redirect" title="Albertus Soegijopranoto">Albertus Soegijopranoto</a>, the first Indonesian bishop. Albertus is his baptismal name, while Soegijopranoto is his traditional Javanese given name.
</p><p>Until now, there is no obligation to have a family name in Indonesia. However, it is becoming more popular for Javanese parents to give family names for their children. The family names are derived from paternal given names.
</p>

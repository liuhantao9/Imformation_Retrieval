<p>The <b>1st parallel north</b> is a <a href="/wiki/Circle_of_latitude" title="Circle of latitude">circle of latitude</a> that is 1 <a href="/wiki/Degree_(angle)" title="Degree (angle)">degree</a> <a href="/wiki/True_north" title="True north">north</a> of the <a href="/wiki/Earth" title="Earth">Earth's</a> <a href="/wiki/Equator" title="Equator">equatorial plane</a>.  It crosses the <a href="/wiki/Atlantic_Ocean" title="Atlantic Ocean">Atlantic Ocean</a>, <a href="/wiki/Africa" title="Africa">Africa</a>, the <a href="/wiki/Indian_Ocean" title="Indian Ocean">Indian Ocean</a>, <a href="/wiki/Southeast_Asia" title="Southeast Asia">Southeast Asia</a>, the <a href="/wiki/Pacific_Ocean" title="Pacific Ocean">Pacific Ocean</a> and <a href="/wiki/South_America" title="South America">South America</a>.
</p><p>The parallel defines part of the border between <a href="/wiki/Equatorial_Guinea" title="Equatorial Guinea">Equatorial Guinea</a> and <a href="/wiki/Gabon" title="Gabon">Gabon</a>.
</p>

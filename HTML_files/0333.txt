<p>The <b>meridian 58° west of <a href="/wiki/Prime_Meridian" class="mw-redirect" title="Prime Meridian">Greenwich</a></b> is a line of <a href="/wiki/Longitude" title="Longitude">longitude</a> that extends from the <a href="/wiki/North_Pole" title="North Pole">North Pole</a> across the <a href="/wiki/Arctic_Ocean" title="Arctic Ocean">Arctic Ocean</a>, <a href="/wiki/North_America" title="North America">North America</a>, the <a href="/wiki/Atlantic_Ocean" title="Atlantic Ocean">Atlantic Ocean</a>, <a href="/wiki/South_America" title="South America">South America</a>, the <a href="/wiki/Southern_Ocean" title="Southern Ocean">Southern Ocean</a>, and <a href="/wiki/Antarctica" title="Antarctica">Antarctica</a> to the <a href="/wiki/South_Pole" title="South Pole">South Pole</a>.
</p><p>The 58th meridian west forms a <a href="/wiki/Great_circle" title="Great circle">great circle</a> with the <a href="/wiki/122nd_meridian_east" title="122nd meridian east">122nd meridian east</a>.
</p>

	<h1 id="firstHeading" class="firstHeading" lang="en">Scottish Gaelic name</h1>
	
	<div id="bodyContent" class="mw-body-content">
		<div id="siteSub" class="noprint">From Wikipedia, the free encyclopedia</div>
		<div id="contentSub"></div>
		
		
		
		<div id="jump-to-nav"></div>
		<a class="mw-jump-link" href="#mw-head">Jump to navigation</a>
		<a class="mw-jump-link" href="#p-search">Jump to search</a>
		<div id="mw-content-text" lang="en" dir="ltr" class="mw-content-ltr"><div class="mw-parser-output"><p>A formal <a href="/wiki/Scottish_Gaelic" title="Scottish Gaelic">Gaelic language</a> name consists of a given name and a surname. First names are either native or nativized (i.e. borrowed and made to fit <a href="/wiki/Scottish_Gaelic_phonology" title="Scottish Gaelic phonology">the Gaelic sound system</a>). Surnames are generally <a href="/wiki/Patronymic_surname" title="Patronymic surname">patronymic</a>, i.e. they refer to a historical ancestor. The form of a surname varies according to whether its bearer is male (e.g. <i lang="gd" title="Scottish Gaelic language text">MacDhòmhnaill</i> "MacDonald") or female (e.g. <i lang="gd" title="Scottish Gaelic language text">NicDhòmhnaill</i> "MacDonald") though for some surnames the adjectival form of a name such as <i lang="gd" title="Scottish Gaelic language text">Dòmhnallach</i> (adjectival form of MacDonald) can be used for both men and women. However, when used in the female form the first letter is <a href="/wiki/Lenition" title="Lenition">lenited</a> (if possible).
</p>

<h2><span class="mw-headline" id="First_names">First names</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Scottish_Gaelic_name&amp;action=edit&amp;section=1" title="Edit section: First names">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Gaelic first names chiefly hail from 5 linguistic layers, <a href="/wiki/Goidelic_languages" title="Goidelic languages">Goidelic</a> and 4 others, coinciding with the main languages of contact: <a href="/wiki/Latin" title="Latin">Latin</a>, <a href="/wiki/Norse_language" class="mw-redirect" title="Norse language">Norse</a>, <a href="/wiki/Anglo-Norman_language" title="Anglo-Norman language">Anglo-Norman</a> and <a href="/wiki/Scots_language" title="Scots language">Scots</a>.<sup id="cite_ref-CGS_1-0" class="reference"><a href="#cite_note-CGS-1">&#91;1&#93;</a></sup> Unusually, male first names outnumber female first names by about a factor of 2:1.<sup id="cite_ref-Morgan_2-0" class="reference"><a href="#cite_note-Morgan-2">&#91;2&#93;</a></sup>
</p>
<h3><span class="mw-headline" id="Goidelic_names">Goidelic names</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Scottish_Gaelic_name&amp;action=edit&amp;section=2" title="Edit section: Goidelic names">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>This layer can be broadly split into three main types:<sup id="cite_ref-CGS_1-1" class="reference"><a href="#cite_note-CGS-1">&#91;1&#93;</a></sup>
</p>
<ul><li>descriptive names (nouns or adjectives), e.g. <i lang="gd" title="Scottish Gaelic language text">Fionn</i> "fair, bright", <i lang="gd" title="Scottish Gaelic language text">Art</i> "bear"</li>
<li>old compounds (names which had fused to the extent of now being opaque); e.g., <i lang="gd" title="Scottish Gaelic language text">Dòmhnall</i> "Donald" (*Dumnoualos "world strength"), <i lang="gd" title="Scottish Gaelic language text">Murchadh</i> "Murdo" (*Moricatus "sea battle")</li>
<li>compounds, e.g. <i lang="gd" title="Scottish Gaelic language text">Donn-slèibhe</i> "Dunlevy" ("brown one of the mountain"), <i lang="gd" title="Scottish Gaelic language text">Gille Chrìost</i> "Gilchrist" ("servant of Christ")</li></ul>
<p>The first two categories were no longer productive for the most part towards the end of the <a href="/wiki/Old_Irish" title="Old Irish">Old Irish</a> period but the last type persisted, reinforced by the coinage of ecclesiastical names following Christianization.<sup id="cite_ref-CGS_1-2" class="reference"><a href="#cite_note-CGS-1">&#91;1&#93;</a></sup>
</p>
<h3><span class="mw-headline" id="Norse_names">Norse names</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Scottish_Gaelic_name&amp;action=edit&amp;section=3" title="Edit section: Norse names">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Quite a number of names still common hail from the period of Norse contact:<sup id="cite_ref-CGS_1-3" class="reference"><a href="#cite_note-CGS-1">&#91;1&#93;</a></sup><sup id="cite_ref-Morgan_2-1" class="reference"><a href="#cite_note-Morgan-2">&#91;2&#93;</a></sup>
</p>
<ul><li><i lang="gd" title="Scottish Gaelic language text">Somhairle</i> &lt; <i lang="non" title="Old Norse language text">Sumarliðr</i></li>
<li><i lang="gd" title="Scottish Gaelic language text">Tormod</i> &lt; <i lang="non" title="Old Norse language text">Þórmóðr</i></li>
<li><i lang="gd" title="Scottish Gaelic language text">Torcuil</i> &lt;  <i lang="non" title="Old Norse language text"><a href="/wiki/Thorkel" title="Thorkel">Þórkell, Þórketill</a></i></li>
<li><i lang="gd" title="Scottish Gaelic language text">Ìomhar</i> &lt; <i lang="non" title="Old Norse language text"><a href="/wiki/Ivar" title="Ivar">Ívarr</a></i></li></ul>
<h3><span class="mw-headline" id="Anglo-Norman">Anglo-Norman</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Scottish_Gaelic_name&amp;action=edit&amp;section=4" title="Edit section: Anglo-Norman">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Names from this layer include:<sup id="cite_ref-CGS_1-4" class="reference"><a href="#cite_note-CGS-1">&#91;1&#93;</a></sup><sup id="cite_ref-Morgan_2-2" class="reference"><a href="#cite_note-Morgan-2">&#91;2&#93;</a></sup>
</p>
<ul><li><i lang="gd" title="Scottish Gaelic language text">Sìleas</i> &lt; <i lang="xno" title="Anglo-Norman language text">Giles</i></li></ul>
<h3><span class="mw-headline" id="Scots">Scots</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Scottish_Gaelic_name&amp;action=edit&amp;section=5" title="Edit section: Scots">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Scots names which have been borrowed into Gaelic include:<sup id="cite_ref-CGS_1-5" class="reference"><a href="#cite_note-CGS-1">&#91;1&#93;</a></sup><sup id="cite_ref-Morgan_2-3" class="reference"><a href="#cite_note-Morgan-2">&#91;2&#93;</a></sup>
</p>
<ul><li><i lang="gd" title="Scottish Gaelic language text">Eairrdsidh</i> &lt; <i lang="sco" title="Scots language text">Archie</i></li></ul>
<h3><span class="mw-headline" id="Latin">Latin</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Scottish_Gaelic_name&amp;action=edit&amp;section=6" title="Edit section: Latin">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Names which were borrowed from Latin include:<sup id="cite_ref-Morgan_2-4" class="reference"><a href="#cite_note-Morgan-2">&#91;2&#93;</a></sup>
</p>
<ul><li><i lang="gd" title="Scottish Gaelic language text">Pàdraig</i> &lt; <i lang="la" title="Latin language text">Patricius</i></li>
<li><i lang="gd" title="Scottish Gaelic language text">Sìle</i> &lt; <i lang="la" title="Latin language text">Caecilia</i></li></ul>
<h3><span id="Borrowing_into_English.2FScots"></span><span class="mw-headline" id="Borrowing_into_English/Scots">Borrowing into English/Scots</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Scottish_Gaelic_name&amp;action=edit&amp;section=7" title="Edit section: Borrowing into English/Scots">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<table class="box-Unreferenced_section plainlinks metadata ambox ambox-content ambox-Unreferenced" role="presentation"><tbody><tr><td class="mbox-image"><div style="width:52px"><a href="/wiki/File:Question_book-new.svg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/en/thumb/9/99/Question_book-new.svg/50px-Question_book-new.svg.png" decoding="async" width="50" height="39" srcset="//upload.wikimedia.org/wikipedia/en/thumb/9/99/Question_book-new.svg/75px-Question_book-new.svg.png 1.5x, //upload.wikimedia.org/wikipedia/en/thumb/9/99/Question_book-new.svg/100px-Question_book-new.svg.png 2x" data-file-width="512" data-file-height="399" /></a></div></td><td class="mbox-text"><div class="mbox-text-span">This section <b>does not <a href="/wiki/Wikipedia:Citing_sources" title="Wikipedia:Citing sources">cite</a> any <a href="/wiki/Wikipedia:Verifiability" title="Wikipedia:Verifiability">sources</a></b>.<span class="hide-when-compact"> Please help <a class="external text" href="https://en.wikipedia.org/w/index.php?title=Scottish_Gaelic_name&amp;action=edit">improve this section</a> by <a href="/wiki/Help:Introduction_to_referencing_with_Wiki_Markup/1" title="Help:Introduction to referencing with Wiki Markup/1">adding citations to reliable sources</a>. Unsourced material may be challenged and <a href="/wiki/Wikipedia:Verifiability#Burden_of_evidence" title="Wikipedia:Verifiability">removed</a>.</span>  <small class="date-container"><i>(<span class="date">April 2017</span>)</i></small><small class="hide-when-compact"><i> (<a href="/wiki/Help:Maintenance_template_removal" title="Help:Maintenance template removal">Learn how and when to remove this template message</a>)</i></small></div></td></tr></tbody></table>
<p>A fair number of Gaelic names were borrowed into English or <a href="/wiki/Scots_language" title="Scots language">Scots</a> at different periods (e.g. Kenneth, Duncan, Donald, Malcolm, Calum, Lachlan, Alasdair, Iain, Eilidh), although it can sometimes be difficult to tell if the donor language was <a href="/wiki/Irish_language" title="Irish language">Irish</a> or Scottish Gaelic (e.g. Deirdre, Rory, Kennedy, Bridget/Bride, Aiden). On occasion, the same name was borrowed more than once due to misinterpretation of Gaelic <a href="/wiki/Morphology_(linguistics)" title="Morphology (linguistics)">morphology</a>. For example, the names Hamish and Mhairi <span class="nowrap"><span class="IPA nopopups noexcerpt"><a href="/wiki/Help:IPA/English" title="Help:IPA/English">/<span style="border-bottom:1px dotted"><span title="/ˈ/: primary stress follows">ˈ</span><span title="&#39;v&#39; in &#39;vie&#39;">v</span><span title="/ɑː/: &#39;a&#39; in &#39;father&#39;">ɑː</span><span title="&#39;r&#39; in &#39;rye&#39;">r</span><span title="/i/: &#39;y&#39; in &#39;happy&#39;">i</span></span>/</a></span></span> are derived from Gaelic <i lang="gd" title="Scottish Gaelic language text">Seumas</i> <small></small><span title="Representation in the International Phonetic Alphabet (IPA)" class="IPA"><a href="/wiki/Help:IPA/Scottish_Gaelic" title="Help:IPA/Scottish Gaelic">[ˈʃeːməs̪]</a></span> and <i lang="gd" title="Scottish Gaelic language text">Màiri</i> <small></small><span title="Representation in the International Phonetic Alphabet (IPA)" class="IPA"><a href="/wiki/Help:IPA/Scottish_Gaelic" title="Help:IPA/Scottish Gaelic">[ˈmaːɾʲɪ]</a></span> but rather than borrowing the root forms, the English/Scots forms are based on the Gaelic <a href="/wiki/Vocative_case" title="Vocative case">vocative case</a> forms <i lang="gd" title="Scottish Gaelic language text">Sheumais</i> <small></small><span title="Representation in the International Phonetic Alphabet (IPA)" class="IPA"><a href="/wiki/Help:IPA/Scottish_Gaelic" title="Help:IPA/Scottish Gaelic">[ˈheːmɪʃ]</a></span> and <i lang="gd" title="Scottish Gaelic language text">Mhàiri</i> <small></small><span title="Representation in the International Phonetic Alphabet (IPA)" class="IPA"><a href="/wiki/Help:IPA/Scottish_Gaelic" title="Help:IPA/Scottish Gaelic">[ˈvaːɾʲɪ]</a></span>.
</p><p>Some names which did not acquire currency outside the Gaelic-speaking world were roughly transliterated into English, such as <i lang="gd" title="Scottish Gaelic language text">Gorm(sh)uil</i> which is often rendered as "Gormelia".
</p><p>Others were with no <a href="/wiki/Cognate" title="Cognate">cognate</a> were often equated with English/Scots names which bore some similarity to the Gaelic name in order to obtain "English equivalents". This includes <i lang="gd" title="Scottish Gaelic language text">Oighrig</i> which was equated with <a href="/wiki/Euphemia" title="Euphemia">Euphemia</a>, <i lang="gd" title="Scottish Gaelic language text">Dìorbhail</i> with <a href="/wiki/Dorothy_(given_name)" title="Dorothy (given name)">Dorothy</a>, <i lang="gd" title="Scottish Gaelic language text">Beathag</i> with <a href="/wiki/Rebecca" title="Rebecca">Rebecca</a> or <a href="/wiki/Sophie" title="Sophie">Sophie</a>.
</p>
<h2><span class="mw-headline" id="Surnames">Surnames</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Scottish_Gaelic_name&amp;action=edit&amp;section=8" title="Edit section: Surnames">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The majority of Gaelic surnames in the Highlands and western parts are <a href="/wiki/Patronymic" title="Patronymic">patronymic</a> in nature and of Goidelic extraction, although epithets, geography or occupation and borrowings also occur in some surnames.<sup id="cite_ref-CGS_1-6" class="reference"><a href="#cite_note-CGS-1">&#91;1&#93;</a></sup> However, many surnames are derived from topographical features or place names, Such surnames include Caddenhead/<i lang="gd" title="Scottish Gaelic language text">Cionnchadach</i>, Cadell/<i lang="gd" title="Scottish Gaelic language text">Cadalach</i>, Cleghorne/<i lang="gd" title="Scottish Gaelic language text">Clagarnach</i>, Dalzell/<i lang="gd" title="Scottish Gaelic language text">Dailghileach</i>, Dalrimple/<i lang="gd" title="Scottish Gaelic language text">Ruimpealach</i>, Elphin/<i lang="gd" title="Scottish Gaelic language text">Ailbhinneach</i>, Inverbervie/<i lang="gd" title="Scottish Gaelic language text">Biorbhach</i>, Kelty/<i lang="gd" title="Scottish Gaelic language text">Cailtidheach</i>, Learmonth/<i lang="gd" title="Scottish Gaelic language text">Learmonadhach</i>, Ochiltree/<i lang="gd" title="Scottish Gaelic language text">Ochailtreabhach</i> and many more. 
</p><p>Campbell/<i lang="gd" title="Scottish Gaelic language text">Caimbeul</i> "crooked mouth" and Cameron/<i lang="gd" title="Scottish Gaelic language text">Camshron</i> "crooked nose" are two examples of surnames based on epithets, while Fraser/<i lang="gd" title="Scottish Gaelic language text">Frisealach</i> is an example of a borrowing (from Anglo-French <i lang="xno" title="Anglo-Norman language text">Fresel</i>).<sup id="cite_ref-CGS_1-7" class="reference"><a href="#cite_note-CGS-1">&#91;1&#93;</a></sup>
</p><p>The usage of patronymic surnames was much more varied than is generally assumed. Historically, clan surnames were used by the descendants or dependants of an ancestor but not generally by everyone in the clan territory.<sup id="cite_ref-CGS_1-8" class="reference"><a href="#cite_note-CGS-1">&#91;1&#93;</a></sup> Only with the advent of a non-Gaelic speaking administration were clan surnames applied en-masse to people in a clan's territory.<sup id="cite_ref-CGS_1-9" class="reference"><a href="#cite_note-CGS-1">&#91;1&#93;</a></sup>
</p>
<h3><span class="mw-headline" id="Formation">Formation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Scottish_Gaelic_name&amp;action=edit&amp;section=9" title="Edit section: Formation">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Patronymic surnames for men feature either the <i lang="gd" title="Scottish Gaelic language text">mac</i> (e.g. <i lang="gd" title="Scottish Gaelic language text">MacDhomhnaill</i>) element or the <a href="/wiki/Nominalization" title="Nominalization">nominalizing</a> suffix <i lang="gd"><a href="https://en.wiktionary.org/wiki/-ach#Scottish_Gaelic" class="extiw" title="wikt:-ach">-ach</a></i> (e.g. <i lang="gd" title="Scottish Gaelic language text">Domhnallach</i>). In the case of women, the element <i lang="gd"><a href="https://en.wiktionary.org/wiki/nic#Scottish_Gaelic" class="extiw" title="wikt:nic">nic</a></i> is used (derived from <i lang="gd"><a href="https://en.wiktionary.org/wiki/nighean#Scottish_Gaelic" class="extiw" title="wikt:nighean">nighean mhic</a></i> "the daughter of the son of").<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup><sup id="cite_ref-5" class="reference"><a href="#cite_note-5">&#91;a&#93;</a></sup> Various other morphological changes (such as <a href="/wiki/Lenition" title="Lenition">lenition</a> or <a href="/wiki/Palatalization_(phonetics)" title="Palatalization (phonetics)">slenderization</a>) may apply in Gaelic, so the surname MacDonald for example may appear as <i lang="gd" title="Scottish Gaelic language text">MacDhòmhnaill</i>, <i lang="gd" title="Scottish Gaelic language text">MhacDhòmhnaill</i>, <i lang="gd" title="Scottish Gaelic language text">MhicDhòmhnaill</i>, <i lang="gd" title="Scottish Gaelic language text">NicDhòmhnaill</i> depending on the grammatical context.
</p><p>As a result of misspellings, one Gaelic surname often corresponds to numerous English/Scots forms, e.g. <i lang="gd" title="Scottish Gaelic language text">MacDhonnchaidh</i> "son of Duncan" may appear as: <i>Donagh(y), Donnagh, Dono(u)gh, MacConachie, MacConachy, MacConaghy, MacConchy, MacConechie, MacConkey, MacConnachie, MacConnechie, MacConnichie, MacConochie, MacConoughy, MacDona, MacDonachie, MacDonachy, MacDonaghy, MacDonaugh, MacDonnach, MacDonnagh, MacDonnoghie, MacDonogh, MacDonoghue, MacDonough, MacDunphy, MacKonochie, MacOnachie, MacOnechy, MacOnochie, Donohue</i> or <i>Donohoe</i> (ignoring the Mac/Mc variation).
</p><p>Scottish Gaelic does not put a space between the <i lang="gd" title="Scottish Gaelic language text">Mac/Nic</i> and the second element, whereas in Irish, there is a space:<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">&#91;5&#93;</a></sup><sup id="cite_ref-7" class="reference"><a href="#cite_note-7">&#91;6&#93;</a></sup>
</p>

<h2><span class="mw-headline" id="Nicknames">Nicknames</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Scottish_Gaelic_name&amp;action=edit&amp;section=10" title="Edit section: Nicknames">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/Nickname" title="Nickname">Nicknames</a> (<a href="/wiki/Scottish_Gaelic_language" class="mw-redirect" title="Scottish Gaelic language">Scottish Gaelic</a>: <i lang="gd">far-ainm</i>, <i lang="gd" title="Scottish Gaelic language text">frith-ainm</i>) in Gaelic operate similarly to those in other languages and usually indicate a physical characteristic, an occupation, a location or an incident the person is associated and so forth.<sup id="cite_ref-HS_8-0" class="reference"><a href="#cite_note-HS-8">&#91;7&#93;</a></sup>

<h2><span class="mw-headline" id="Identifying_names">Identifying names</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Scottish_Gaelic_name&amp;action=edit&amp;section=17" title="Edit section: Identifying names">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Due to the relative paucity of names and surnames in Gaelic, the official name of a person (i.e. first name plus a surname, in Gaelic or English/Scots) is rarely used in Gaelic speaking communities as, with a small number of surnames usually predominating in an area, there are usually several people who go by the same combination, for example John MacLeod might apply to several people in the same village.<sup id="cite_ref-CGS_1-10" class="reference"><a href="#cite_note-CGS-1">&#91;1&#93;</a></sup><sup id="cite_ref-HS_8-1" class="reference"><a href="#cite_note-HS-8">&#91;7&#93;</a></sup> In everyday life, this is usually solved by using the first name of a man, followed by the first name of his father in the <a href="/wiki/Genitive_case" title="Genitive case">genitive case</a> or by using the first name plus an <a href="/wiki/Epithet" title="Epithet">epithet</a>.<sup id="cite_ref-HS_8-2" class="reference"><a href="#cite_note-HS-8">&#91;7&#93;</a></sup> So a man called James (<i lang="gd" title="Scottish Gaelic language text">Seumas</i>) with a father called Neil (<i lang="gd" title="Scottish Gaelic language text">Niall</i>) would become <i lang="gd" title="Scottish Gaelic language text">Seumas Nèill</i> or <i lang="gd" title="Scottish Gaelic language text">Seumas a' Ghlinne</i> ("James of the glen").<sup id="cite_ref-HS_8-3" class="reference"><a href="#cite_note-HS-8">&#91;7&#93;</a></sup> In the case of married women, the convention is normally to use <i lang="gd" title="Scottish Gaelic language text">bean</i> ("wife") plus the husband's first name and father's first name, in our example resulting in <i lang="gd" title="Scottish Gaelic language text">Bean Sheumais Nèill</i> ("the wife of Neil's James"). <div style="clear:both;"></div>
<p>Historically, such an identifying name would take the <i lang="gd" title="Scottish Gaelic language text">mac</i> "son" element, e.g. <i lang="gd" title="Scottish Gaelic language text">Dòmhnall mac Sheumais mac Nèill</i> ("Donald son of James son of Neill") but in modern usage, this is usually dropped, resulting in <i lang="gd" title="Scottish Gaelic language text">Dòmhnall Sheumais Nèill</i>.<sup id="cite_ref-CGS_1-11" class="reference"><a href="#cite_note-CGS-1">&#91;1&#93;</a></sup>
</p><p>Identifying names sometimes use female reference points, for example if a local woman marries an outsider, this may result in the children being identified via the mother. <i lang="gd" title="Scottish Gaelic language text">Dòmhnall Chiorstan</i> ("Kirsten's Donald") for instance would indicate a son called Donald with a mother called Kirsten.<sup id="cite_ref-16" class="reference"><a href="#cite_note-16">&#91;15&#93;</a></sup>
</p>

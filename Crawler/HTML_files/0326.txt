<p><b>Ukrainian names</b> are given names that originated in <a href="/wiki/Ukraine" title="Ukraine">Ukraine</a>. In addition to the <a href="/wiki/Given_name" title="Given name">given names</a>, <a href="/wiki/Ukrainians" title="Ukrainians">Ukrainians</a> also have  <a href="/wiki/Patronymic" title="Patronymic">patronymic</a> and <a href="/wiki/Surname" title="Surname">family names</a> (surnames; see: <i><a href="/wiki/Ukrainian_surnames" title="Ukrainian surnames">Ukrainian surnames</a></i>).
</p>

<h2><span class="mw-headline" id="Ukrainian_given_names">Ukrainian given names</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ukrainian_name&amp;action=edit&amp;section=1" title="Edit section: Ukrainian given names">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/Diminutive" title="Diminutive">Diminutive</a> and <a href="/wiki/Hypocoristic" class="mw-redirect" title="Hypocoristic">hypocoristic</a> forms native to <a href="/wiki/Ukrainian_language" title="Ukrainian language">Ukrainian language</a> have either empty <a href="/wiki/Inflexion" class="mw-redirect" title="Inflexion">inflexional</a> <a href="/wiki/Suffix" title="Suffix">Suffix</a> (<span lang="uk" title="Ukrainian language text">Гринь</span>, <span lang="uk" title="Ukrainian language text">Сень</span>) or affixes -o, -e, -ik (<span lang="uk" title="Ukrainian language text">Славко</span>, <span lang="uk" title="Ukrainian language text">Грицько</span>, <span lang="uk" title="Ukrainian language text">Славця</span>, <span lang="uk" title="Ukrainian language text">Грицуня</span>, <span lang="uk" title="Ukrainian language text">Андрійчик</span>, <span lang="uk" title="Ukrainian language text">Петрик</span>).
</p><p>As with most cultures, a person has a given name chosen by the parents. First names in East-<a href="/wiki/Slavic_languages" title="Slavic languages">Slavic languages</a> mostly originate from two sources: <a href="/wiki/Eastern_Orthodox_Church" title="Eastern Orthodox Church">Orthodox</a> church tradition (which is itself of <a href="/wiki/Greek_language" title="Greek language">Greek</a> origin), <a href="/wiki/Eastern_Catholic_Churches" title="Eastern Catholic Churches">Catholic</a> church tradition (which is itself of <a href="/wiki/Latin_language" class="mw-redirect" title="Latin language">Latin</a> origin) and <a href="/wiki/Slavic_names" title="Slavic names">native pre-Christian Slavic origin lexicons</a>. Most names have several diminutive forms.
</p>

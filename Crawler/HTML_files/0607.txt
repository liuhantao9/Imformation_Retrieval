<p><b>Huddersfield bus station</b> serves the town of <a href="/wiki/Huddersfield" title="Huddersfield">Huddersfield</a>, <a href="/wiki/West_Yorkshire" title="West Yorkshire">West Yorkshire</a>, <a href="/wiki/England" title="England">England</a>.
</p><p>The bus station was opened on Sunday 1 December 1974 and is owned and managed by <a href="/wiki/West_Yorkshire_Metro" title="West Yorkshire Metro">Metro</a>. It is the busiest bus station in West Yorkshire and is used by more than 33,000 passengers every day<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup>. The bus station is situated in Huddersfield town centre, underneath the <a href="/wiki/Multi-storey_car_park" class="mw-redirect" title="Multi-storey car park">Multi-storey car park</a>. It is bordered by the Ring Road (<a href="/wiki/A62_road" title="A62 road">Castlegate A62</a>) and can be accessed from High Street, Upperhead Row and Henry Street.
</p><p>There are 25 pick-up and three alighting only stands at the bus station.
</p>
<h2><span class="mw-headline" id="Services">Services</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Huddersfield_bus_station&amp;action=edit&amp;section=1" title="Edit section: Services">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The main operators at the bus station include <a href="/wiki/First_West_Yorkshire" title="First West Yorkshire">First West Yorkshire</a>, <a href="/wiki/Arriva_Yorkshire" title="Arriva Yorkshire">Arriva Yorkshire</a> and <a href="/wiki/Yorkshire_Tiger" title="Yorkshire Tiger">Yorkshire Tiger</a> with other services run by <a href="/wiki/First_Greater_Manchester" title="First Greater Manchester">First Greater Manchester</a>,TLC, <a href="/wiki/Yorkshire_Tiger#Tiger_Blue" title="Yorkshire Tiger">Tiger Blue</a> and <a href="/wiki/Stotts_Coaches" title="Stotts Coaches">Stotts Coaches</a>. <a href="/wiki/National_Express_Coaches" title="National Express Coaches">National Express</a> coaches run nationwide from here.
</p><p>In March 2009, <a href="/wiki/Megabus_(Europe)" title="Megabus (Europe)">Megabus</a> started to run a coach service, called <a href="/wiki/Megabus_(Europe)#Megabusplus" title="Megabus (Europe)">Megabusplus</a> to <a href="/wiki/London" title="London">London</a>, which involves changing from the coach on to an <a href="/wiki/East_Midlands_Railway" title="East Midlands Railway">East Midlands Railway</a> service at <a href="/wiki/East_Midlands_Parkway_railway_station" title="East Midlands Parkway railway station">East Midlands Parkway</a>.
</p><p>There are plenty of local services which link Huddersfield with the surrounding areas of the town such as <a href="/wiki/Denby_Dale" title="Denby Dale">Denby Dale</a>, <a href="/wiki/Golcar" title="Golcar">Golcar</a>, <a href="/wiki/Holmfirth" title="Holmfirth">Holmfirth</a>, <a href="/wiki/Lindley,_West_Yorkshire" class="mw-redirect" title="Lindley, West Yorkshire">Lindley</a>, <a href="/wiki/Marsden,_West_Yorkshire" title="Marsden, West Yorkshire">Marsden</a>, <a href="/wiki/Meltham" title="Meltham">Meltham</a>, <a href="/wiki/Milnsbridge" title="Milnsbridge">Milnsbridge</a>, <a href="/wiki/Slaithwaite" title="Slaithwaite">Slaithwaite</a>. In addition to this, there are several services which run to other towns in West Yorkshire such as <a href="/wiki/Batley" title="Batley">Batley</a>, <a href="/wiki/Bradford" title="Bradford">Bradford</a>, <a href="/wiki/Brighouse" title="Brighouse">Brighouse</a>, <a href="/wiki/Cleckheaton" title="Cleckheaton">Cleckheaton</a>, <a href="/wiki/Dewsbury" title="Dewsbury">Dewsbury</a>, <a href="/wiki/Elland" title="Elland">Elland</a>, <a href="/wiki/Halifax,_England" class="mw-redirect" title="Halifax, England">Halifax</a>, <a href="/wiki/Hebden_Bridge" title="Hebden Bridge">Hebden Bridge</a>, <a href="/wiki/Heckmondwike" title="Heckmondwike">Heckmondwike</a>, <a href="/wiki/Leeds" title="Leeds">Leeds</a>, <a href="/wiki/Mirfield" title="Mirfield">Mirfield</a>, <a href="/wiki/Morley,_West_Yorkshire" title="Morley, West Yorkshire">Morley</a> and <a href="/wiki/Wakefield,_England" class="mw-redirect" title="Wakefield, England">Wakefield</a>, the <a href="/wiki/White_Rose_Shopping_Centre" class="mw-redirect" title="White Rose Shopping Centre">White Rose Shopping Centre</a> plus the <a href="/wiki/Greater_Manchester_bus_route_184" title="Greater Manchester bus route 184">184</a> service which links Huddersfield with <a href="/wiki/Manchester" title="Manchester">Manchester</a> and <a href="/wiki/Oldham" title="Oldham">Oldham</a>.
</p>
